# 📄 RAG PDF Chatbot

An AI-powered Retrieval-Augmented Generation (RAG) chatbot that allows users to upload PDF documents and ask questions based on their content. The chatbot uses Google Gemini for response generation, ChromaDB for vector storage, and Sentence Transformers for embeddings.

## 🚀 Features

- 📄 Upload PDF documents
- ✂️ Automatic text extraction and chunking
- 🧠 Semantic search using embeddings
- 🔍 Retrieval-Augmented Generation (RAG)
- 🤖 Google Gemini-powered answers
- ⚡ Fast document indexing with ChromaDB
- 🖥️ Interactive Gradio interface

## 🛠️ Technologies Used

- Python
- Google Gemini API
- ChromaDB
- Sentence Transformers
- PyPDF
- Gradio
